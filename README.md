# Tomcat简易版 V1.0
## 项目介绍 
    基于《how tomcat works》(中文版:《深入剖析Tomcat》)每个章节的内容（ZAO LUN ZI）
    本版本实现了最基本的HTML等静态资源文件的server容器
## 快速开始
    1. 将所需要使用的静态HTML文件放入resources/webroot/路径下
    2. 运行HttpServer.main()方法启动服务
    3. 打开浏览器输入127.0.0.1:8080/{你的资源文件}即可访问静态HTML资源
## 运行环境
    - java 8
    - maven 3.0+
    - chrome
